# ROOT: Foundation Consulting

ROOT is a premium consulting service dedicated to structural and psychological re-alignment. We help individuals and organizations find their essence and build foundations that don't crumble under pressure.

## Our Philosophy

We believe that every problem has a "root" cause. In a world that focuses on symptomatic fixes, we dive deep into the core structures to facilitate genuine transformation.

## Key Offerings

- **ROOT 1:1 Consulting**: Precise diagnostic and re-alignment of individual thought processes and priorities.
- **ROOTED: 4-Week Program**: A comprehensive course designed to build an unshakable inner foundation.

## Tech Stack

- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
